# Testování REST API

- nepoužívá se GUI, komunikace pomocí textově orientovaných formátů (JSON)

## API

- Application Programming Interface
- určuje, jak mají SW komponenty komunikovat mezi sebou (rozhraní)
- různé druhy - REST, SOAP, ...

### Web Service

- webová HTTP služba v síti
- umožňuje jiným systémům komunikaci s ní pomocí definovaného způsobu (REST, SOAP)
- charakteristiky
	- volná vazba (neměnné rozhraní)
	- synchronní a asynchronní komunikace
	- výměna dokumentů

### SOAP

- Simple Object Access Protocol - striktně definovaný protokol
- vstup (request) - SOAP Message XML
- výstup (response) - XML dokument
- testování: práce s XML a jeho odeslání přes POST

### REST

- Representational State Transfer - architektonický styl s velmi volnými pravidly
- parametry v URI požadavku, nestavovost
- odpověď libovolná - JSON / XML / plain text
- různé metody pro manipulaci se zdrojem
- testování: manipulace s URL a různé metody

## curl

- nejznámější a nejpoužívanější nástroj pro přesuny dat ze serveru a na server
- podpora velkého množství protokolů
- používán v příkazové řádce

## SoapUI

- nástroj na testování SOAP a REST API (funkční, regresní, zátěžové, bezpečnostní)
- pro REST modernější nástroje - Postman, Insomnia
- podpora Test Suite, Test Case a Test Step, proměnné, Groovy scripty, ...
