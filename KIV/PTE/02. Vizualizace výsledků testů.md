# Vizualizace výsledků testů

- report unit testů v IDE vhodný pro vývojáře
- jiné testy dle testovacího frameworku, většinou přizpůsobitelný testerům
- RobotFramework - vhodný report pro všechny, detailní pro testery

cíl výsledků testů
- souhrnné, přehledné, průběžné
- možnost volby detailů, filtrů, historie

### Princip řešení

- průběžné posílání výsledků testů do jiného systému (pomocí listeneru nebo v CI/CD)
- jednotný formát pro report výsledků testů (především agregovatelná data, bez detailů)
- výsledky ukládány a vizualizovány na jednom místě (Elasticsearch + Kibana)

### Elasticsearch

- úložiště dat, mnoho formátů
- RESTful rozhraní s propracovaným REST API
- vysoká dostupnost, rychlost, škálovatelnost
- multiplatformní + Docker

### Kibana

- sofistikovaná vizualizace dat
- přímá podpora Elasticsearch, možnost procházení dat
- vizualizace probíhá z dat získávaných v reálném čase
- flexibilita filtrů, vizualizací a dashboardů
- multiplatformní + Docker
